# react-responcy
A React component for responsive source calls.

### Usage

```javascript
<Responcy
  src='[{"size": "300", "src": "<URL>"}, 
          {"size": "400", "src":"<URL>"},
            {"size": "500", "src":"<URL>"}]'>
</Responcy>
```

You need to specify your source definitions and ready to bake!
